Naming convention adalah aturan Java dalam memberi nama pada variabel atau class supaya kode lebih mudah dibaca. Di Java biasanya nama class diawali huruf besar atau disebut PascalCase convention (UpperCamelCase), sedangkan variabel menggunakan huruf kecil di awal atau disebut camelCase convention, contoh penamaan variabel yang saya kerjakan di latihan yaitu namaProduk atau hargaSatuan.

Penamaan variabel menurut saya cukup penting karena bisa membantu memahami fungsi kode. Kalau nama variabelnya jelas, kita bisa langsung tahu kegunaannya tanpa harus membaca seluruh program.

Operator aritmatika di Java ada +, -, *, /, dan %. Operator tersebut digunakan untuk melakukan perhitungan sederhana, misalnya menghitung total nilai stok dari harga satuan dan jumlah stok. Di LatihanKalkulator juga terlihat dua angka bisa saya proses dengan berbagai operasi yang ada.

Di LatihanKalkulator, saya juga menggunakan tipe data double pada hasilDesimal = (double) angka1 / angka2. Karena kalau dua nilai int dibagi, hasilnya hanya bilangan bulat dan tidak akan menampilkan hasil desimal. Dengan mengubah salah satu nilai menjadi double, hasil pembagian bisa menampilkan angka desimal sehingga hasilnya sesuai dengan yang diinginkan.
